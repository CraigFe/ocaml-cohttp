Module Cohttp_lwt__.Server

module Make : functor (IO : Cohttp_lwt.S.IO) -> Cohttp_lwt.S.Server with module IO = IO

The Make functor glues together a Cohttp.S.IO implementation to send requests down a connection that is established by the Net module. The resulting module satisfies the Server module type.
